# [LootBoxAPI](https://lootbox.eu)
An unofficial Overwatch API System.


### Installation
The api requires [Node.js](https://nodejs.org/) v4+ to run.


```sh
$ git clone https://gitlab.com/SingularityIO/LootBoxAPI.git api
$ cd api
$ npm install
$ node App.js
```
